# URL Shortner

This project is a URL shortener built using:

- Node.js
- Express
- Typescript
- Mongodb
- Mongoose

## How to use 

### Clone this repository

```
git clone https://github.com/Reingaw/DIO-URL-shortener.git
```

### Install dependencies

```
npm install
```
or

```
yarn add
```

### Run the project

```
npm run dev
```

or

```
yarn run dev
```
